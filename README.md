# Vue Pomodoro Timer

A simple fun project for creating a Pomodoro Timer using Vue. The tomato red color displays while it's counting down the time and it changes to a green tomato when your on a break. There is a little ding at the end of each round of counting down.

# vue-pomodoro-timer

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

For a detailed explanation on how things work, check out the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).
